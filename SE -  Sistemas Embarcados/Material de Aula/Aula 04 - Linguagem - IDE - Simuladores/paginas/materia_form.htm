02)  Linguagem de programação:

 

Nós seres humanos precisamos converter as nossas ideias para uma forma que os computadores consigam processar, ou seja, a linguagem de máquina.

 

Os computadores de hoje (ainda) não conseguem entender a linguagem natural que nós usamos no dia a dia, então precisamos de um outro "idioma" especial para instruir o computador a fazer as tarefas que desejamos. Esse "idioma" é uma linguagem de programação, e na verdade existem muitas delas.

 

A linguagem de programação utilizada no Arduino é a linguagem C/C++ (com pequenas modificações), que é uma linguagem muito tradicional e conhecida.

 

Para converter um programa escrito em uma linguagem de alto nível (humana) para linguagem de máquina, nós utilizamos uma coisa chamada compilador. A ação de converter um programa para linguagem de máquina é chamada compilar.

 

Para compilar um programa, normalmente se utiliza um ambiente de desenvolvimento (ou IDE, do inglês Integrated Development Environment), que é um aplicativo de computador que possui um compilador integrado, onde você pode escrever o seu programa e compilá-lo.

 

No caso do Arduino, esse ambiente de desenvolvimento é o Arduino IDE (veremos mais a frente).

 

O texto contendo o programa em uma linguagem de programação de alto nível também é conhecido como o código fonte do programa.

04)  Simuladores e Ferramentas de Diagramação:

 

Para qualquer desenvolvimento em uma determinada linguagem é importante manter uma documentação atualizada e organizada, por motivos já conhecidos.

 

Não seria diferente esses cuidados, se tratando da parte física de nosso projeto de automação com Arduino.

 

É de suma importância, criamos um desenho esquemático detalhando os componentes utilizados e as ligações para que não cometamos erros que possam queimar componentes e até mesmo danificar nossa placa Arduino.

  

Para diagramação (esquema ou desenho esquemático) do seu circuito recomendamos alguns softwares abaixo e seus respectivos links de download:

 

(Fritzing - Pago) https://fritzing.org/home/ (versão 0.9.10 em 22/05/2022)

 

(Kicad – Open Source) https://kicad.org/ (versão estável 6.0.10 em 197/12/2022)

05)  Simuladores de Circuitos:

 

Quando se está trabalhando com eletrônica geralmente não montamos diretamente o circuito, devido a possibilidade de erros de projeto, ocasionando em muitos casos a queima de componentes ou do próprio Arduino.

 

Recomendamos algumas ferramentas para organizar e montar seu projeto para só assim montar fisicamente o circuito em uma protoboard, sabendo que tudo está correto e revisado de forma adequada.

 

Lei de Murphy:

 

“Se alguma coisa pode dar errado, dará. E mais, dará errado da pior maneira, no pior momento e de modo que cause o maior dano possível.”

 

Para simulação do seu circuito eletrônico uma boa solução é o site Tinkercad que pode ser acessado pelo link a seguir:

 

https://www.tinkercad.com/

 

O site é mantido pelo Autodesk empresa líder em ferramentas de modelagem 2D, 3D e CAD como o AutoCad.

 

O acesso é gratuito bastando fazer um cadastro para se adquirir uma conta de login.

 

Fácil de usar e cheio de recurso, que facilitam a simulação dos circuitos com precisão e realismo.
